Jogo de memória desenvolvido com Python e Pygame.

## Como jogar

Clique em duas cartas para tentar encontrar o par.
Se as cartas forem iguais, elas ficam abertas.
Se não forem, elas viram de volta após um instante.
O objetivo é encontrar todos os pares antes do tempo acabar

## Como executar

1. Instale as dependências:
   pip install -r requirements.txt

2. Execute o jogo:
   python main.py
